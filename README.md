# Kenya Base Tours

Kenya Base Tours is a web application for exploring and booking tours in Kenya and nearby destinations. It features destination details, tour packages, transport services, and contact information.

## Features
- Browse destinations and tours
- View detailed information for each destination
- Contact form for inquiries
- Responsive design

## Technologies Used
- React
- CSS

## Getting Started

### Prerequisites
- Node.js
- npm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Muyundo/kenya-base-tours.git
   ```
2. Navigate to the project directory:
   ```bash
   cd kenya-base-tours
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application
```bash
npm start
```
The app will run locally at `http://localhost:3000`.

## Project Structure
- `src/` - Source code
  - `components/` - Reusable UI components
  - `pages/` - Page components
  - `data/` - Data files
  - `assets/` - Images and other assets
  - `styles/` - Global and page styles
- `public/` - Static files
